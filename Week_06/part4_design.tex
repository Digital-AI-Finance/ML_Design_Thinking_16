\small
Create personalized radio experience for 500M users

\vspace{0.3cm}
\textcolor{mlorange}{\textbf{AI Solution:}}
\begin{itemize}
\item Voice synthesis (natural DJ)
\item Music recommendation
\item Context awareness
\item Real-time adaptation
\end{itemize}

\textcolor{mlgreen}{\textbf{Results:}}
\begin{itemize}
\item 2x engagement
\item 30\% longer sessions
\item 4.8 star rating
\end{itemize}

\column{0.48\textwidth}
\textcolor{mlpurple}{\textbf{Design Process}}

\small
\begin{enumerate}
\item User research (2 weeks)
\item AI voice prototypes (1 week)
\item Music curation models (2 weeks)
\item Interface design (1 week)
\item A/B testing (2 weeks)
\item Global rollout (4 weeks)
\end{enumerate}

\textcolor{mlcyan}{\textbf{Key Learning:}}\\
\small AI enables hyper-personalization impossible with human DJs
