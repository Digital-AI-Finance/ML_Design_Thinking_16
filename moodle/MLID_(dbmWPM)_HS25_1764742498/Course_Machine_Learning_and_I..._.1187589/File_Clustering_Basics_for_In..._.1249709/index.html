Description

This week introduces clustering as a way to find hidden patterns and groups in data, which can help identify new opportunities for innovation. Students learn how algorithms like K-means and DBSCAN work, how to choose parameters (e.g. number of clusters, density), and how to detect outliers. We practice using simple datasets and explore how to prepare and scale data. Clustering results are discussed as “opportunity spaces” that can inspire new ideas. The session links the machine learning workflow (prepare → model → test) with the design thinking process (empathize → define → ideate → prototype → test), showing how data-driven insights can support creative innovation.
